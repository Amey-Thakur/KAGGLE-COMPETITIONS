## Overview

This repository features an advanced solution for the **"Are You A Robot?"** Kaggle competition. The goal is to detect and classify AI-generated text using specialized Transformer-based classifiers. The implementation focuses on robust feature engineering, high-performance fine-tuning of pre-trained language models, and precise evaluation of statistical anomalies in machine-authored text.

## Strategic Approach

The detection pipeline utilizes the following techniques:
1.  **State-of-the-Art Fine-Tuning**: Leveraging models like DeBERTa-v3 to learn patterns of machine-generated lexical consistency.
2.  **Structural Fingerprinting**: Analysis of paragraph-level and sentence-level distribution metrics to distinguish robot signatures.
3.  **Threshold Optimization**: Calibrating the classification boundary for maximum F1 performance on competition test sets.
